  # comandos TERMINAIS
1. git init Inicializa um repositório Git na pasta atual.
2. git clone Clona (copia) um repositório remoto para o seu computador.
3. git status Mostra o estado atual dos arquivos (modificados, adicionados, etc.).
4. git add Adiciona arquivos para a área de preparação (staging).
5. git commit Salva as alterações com uma mensagem.
6. git push Envia as alterações para o repositório remoto
7. git pull Atualiza seu repositório local com as mudanças do remoto.
8. git branch Lista ou cria novas branches.
9. git checkout Troca de branch ou restaura arquivos.
10. git merge Mescla mudanças de uma branch em outra.
  COMANDOS DO GIT BASH
1. pwd Mostra o caminho da pasta atual.
2. Lista os arquivos e pastas do diretório atual.
3. cd Muda para uma pasta.
4. cd .. Volta para a pasta anterior.
5. mkdir Cria uma nova pasta.
6. touch Cria um novo arquivo vazio.
7. rm Remove um arquivo.
8. clear Limpa o terminal.
9. code .Abre a pasta no Visual Studio Code.
10. exit Fecha o terminal.
   COMANDOS DE CONTROLE DO GIT
1. git config --list  Mostra todas as configurações atuais do Git.
2. git config --global user.name "Seu Nome"  Define o nome do usuário no Git.
3. git config --global user.email "seu@email.com"  Define o email do usuário no Git.
4. git init  Inicializa um repositório Git.
5. git clone  Clona um repositório remoto.
6. git add .  Adiciona todos os arquivos para a área de preparação (staging).
7. git commit -m "mensagem"  Salva as alterações com uma mensagem.
8. git log  Mostra o histórico de commits.
9. git remote -v  Mostra os repositórios remotos conectados.
10. git push origin main  Envia as alterações para o repositório remoto.
11. git pull origin main  Atualiza o repositório local com o remoto.
12. git branch  Lista as branches.
13. git checkout -b nova-branch  Cria e muda para uma nova branch.
14. git merge nova-branch  Mescla uma branch na atual.
